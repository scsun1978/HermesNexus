# Review

Use this command to review a proposed HermesNexus change.

## Review Criteria

- Matches MVP Phase 1 scope
- Keeps local development separate from deployment validation
- Uses the development/test server `scsun@172.16.100.101:22` where appropriate
- Has clear verification steps
- Avoids unnecessary complexity
- Does not conflict with the project docs

## What to Check

- File paths
- Scope creep
- Missing tests
- Missing verification
- Inconsistent terminology
- Deployment assumptions

## Output Format

- Summary
- Risks
- Missing items
- Suggested fixes
