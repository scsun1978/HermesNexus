# 41 实施里程碑与交付拆分

## 目标

把整个 Hermes 分布式边缘设备管理系统拆成可以逐步交付的实施阶段。

## 实施阶段

### Phase 0：项目骨架
交付：
- 仓库结构
- 配置骨架
- 基础文档
- 目录规范

### Phase 1：单边缘节点 MVP
交付：
- 设备接入
- 基础巡检
- 任务下发
- 任务结果回传

### Phase 2：云端控制平面
交付：
- 资产管理
- 任务编排
- 审计记录
- 基础控制台

### Phase 3：安全与可靠性
交付：
- 节点身份
- mTLS / token
- 审批流
- 回滚机制

### Phase 4：多协议设备接入
交付：
- SSH
- MQTT
- SNMP
- NETCONF / RESTCONF

### Phase 5：知识与自动化
交付：
- 记忆沉淀
- 技能复用
- 告警联动
- 日报周报自动化

## 每阶段的验收原则

- 可运行
- 可观察
- 可回滚
- 可审计
- 可复现

## 交付物建议

- 代码
- 配置
- 部署脚本
- API 文档
- 测试报告
- 运维手册
