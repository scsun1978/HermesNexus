## 技术实现

### 数据库架构
- **数据库类型**: SQLite 3
- **数据库文件**: `./data/hermesnexus.db`
- **表结构设计**: 5个核心表
  - `nodes` - 节点信息表
  - `devices` - 设备信息表
  - `jobs` - 任务信息表
  - `events` - 事件记录表
  - `audit_logs` - 审计日志表

### 核心特性
- **线程安全**: 使用锁机制保证并发安全
- **ACID事务**: 支持原子性、一致性、隔离性、持久性
- **外键约束**: 启用外键约束保证数据完整性
- **WAL模式**: 使用Write-Ahead Logging提高并发性能
- **索引优化**: 为常用查询字段创建索引
