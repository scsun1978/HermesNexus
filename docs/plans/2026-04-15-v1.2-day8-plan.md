## 🎯 Day 8 目标

**目标**: 让批量和多节点操作可追踪

确保所有批量操作都有完整的审计记录，能够追溯操作历史、分析失败原因、统计操作成功率。

## 📋 具体任务清单

### 任务1：批量操作落审计
- [ ] 在批量操作执行时自动记录审计日志
- [ ] 记录操作ID、操作类型、操作时间、操作用户
- [ ] 记录操作参数和操作结果
- [ ] 支持审计数据的持久化存储

### 任务2：成功项 / 失败项分开记录
- [ ] 审计记录中区分成功和失败的项
- [ ] 记录每个项目的详细结果（成功/失败/错误信息）
- [ ] 提供成功率和失败率统计
- [ ] 支持按成功/失败状态过滤审计记录

### 任务3：失败摘要可查询
- [ ] 增强审计查询API，支持按操作ID查询
- [ ] 支持按错误类型查询失败记录
- [ ] 提供失败原因统计分析
- [ ] 支持失败操作的详情查看

### 任务4：关联节点 / 任务 / 资产上下文
- [ ] 审计记录包含关联的节点、任务、资产信息
- [ ] 支持按资产ID查询相关审计记录
- [ ] 支持按节点ID查询相关审计记录
- [ ] 提供审计记录的时间线视图

## 🔧 技术实现方案

### 审计数据模型

#### 批量操作审计记录
```python
class BatchOperationAudit(BaseModel):
    """批量操作审计记录"""
    audit_id: str = Field(..., description="审计ID")
    operation_id: str = Field(..., description="操作ID")
    operation_type: str = Field(..., description="操作类型")
    user_id: Optional[str] = Field(None, description="操作用户ID")
    timestamp: datetime = Field(..., description="操作时间")
    
    # 操作参数
    parameters: Dict[str, Any] = Field(..., description="操作参数")
    
    # 操作结果
    total_items: int = Field(..., description="总项目数")
    successful_items: int = Field(..., description="成功项目数")
    failed_items: int = Field(..., description="失败项目数")
    success_rate: float = Field(..., description="成功率")
    
    # 详细结果
    results: List[BatchItemAudit] = Field(default_factory=list, description="详细结果")
    error_summary: Dict[str, int] = Field(default_factory=dict, description="错误摘要")
    
    # 关联信息
    related_assets: List[str] = Field(default_factory=list, description="关联资产ID")
    related_nodes: List[str] = Field(default_factory=list, description="关联节点ID")
    related_tasks: List[str] = Field(default_factory=list, description="关联任务ID")
```

#### 单个项目审计结果
```python
class BatchItemAudit(BaseModel):
    """批量操作单项审计"""
    item_id: str = Field(..., description="项目ID")
    success: bool = Field(..., description="是否成功")
    operation_type: str = Field(..., description="操作类型")
    error_code: Optional[str] = Field(None, description="错误代码")
    error_message: Optional[str] = Field(None, description="错误消息")
    timestamp: datetime = Field(default_factory=datetime.now, description="操作时间")
    
    # 关联信息
    asset_id: Optional[str] = Field(None, description="关联资产ID")
    node_id: Optional[str] = Field(None, description="关联节点ID")
    task_id: Optional[str] = Field(None, description="关联任务ID")
```

### 审计服务架构

#### 审计服务接口
```python
class AuditService:
    """审计服务"""
    
    async def log_batch_operation(self, operation: BatchOperationResponse, 
                                  user_id: Optional[str] = None) -> BatchOperationAudit
    """记录批量操作审计"""
    
    async def query_batch_audit(self, operation_id: str) -> Optional[BatchOperationAudit]
    """查询批量操作审计"""
    
    async def query_failed_operations(self, error_type: Optional[str] = None, 
                                     start_time: Optional[datetime] = None,
                                     end_time: Optional[datetime] = None) -> List[BatchOperationAudit]
    """查询失败的操作"""
    
    async def query_asset_audit_history(self, asset_id: str) -> List[BatchOperationAudit]
    """查询资产审计历史"""
    
    async def get_operation_statistics(self, start_time: datetime, 
                                      end_time: datetime) -> Dict[str, Any]
    """获取操作统计信息"""
```

### 审计存储方案

#### 内存存储（开发环境）
```python
class InMemoryAuditStorage:
    """内存审计存储"""
    
    def __init__(self):
        self.audit_records: Dict[str, BatchOperationAudit] = {}
        self.asset_index: Dict[str, List[str]] = {}  # asset_id -> audit_ids
        self.error_index: Dict[str, List[str]] = {}  # error_type -> audit_ids
    
    async def save_audit(self, audit: BatchOperationAudit) -> bool
    async def get_audit(self, audit_id: str) -> Optional[BatchOperationAudit]
    async def query_by_asset(self, asset_id: str) -> List[BatchOperationAudit]
    async def query_by_error_type(self, error_type: str) -> List[BatchOperationAudit]
```
