# 50 故障演练与验收清单

## 目标

把系统上线前必须验证的故障和验收项列出来。

## 故障演练项

### 1. 云端不可用
- 边缘是否能继续执行本地任务
- 恢复后是否补传事件

### 2. 边缘节点重启
- 任务是否能恢复
- 本地队列是否保留

### 3. 设备失联
- 告警是否上报
- 是否进入重试与降级

### 4. 凭证失效
- 是否能正确拒绝执行
- 是否能触发轮换流程

### 5. 批量变更失败
- 是否能逐步回滚
- 是否能锁定风险范围

## 验收清单

- 节点注册成功
- 任务下发成功
- 巡检能完成
- 审批流能工作
- 回滚能触发
- 审计可追踪
- 日志可检索
- 断网可自治

## 上线前最低标准

- 至少跑完一轮云边联调
- 至少跑完一次断网演练
- 至少跑完一次回滚演练
- 至少跑完一次证书轮换演练

## 建议

- 演练结果进入知识库
- 每次演练都形成复盘记录
- 演练脚本定期更新
