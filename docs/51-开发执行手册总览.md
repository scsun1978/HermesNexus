# 51 开发执行手册总览

## 目标

把 Hermes 分布式边缘设备管理系统的设计文档，收敛成开发团队可以直接执行的工作手册。

## 手册包含什么

- 仓库初始化方式
- 本地开发启动方式
- 代码规范
- 测试规范
- 联调规范
- 发布与回滚流程
- 验收清单

## 使用方式

1. 先看总览，理解阶段目标。
2. 再看本地开发与仓库结构。
3. 开发前先确认测试与接口契约。
4. 发布前必须跑验收清单。
