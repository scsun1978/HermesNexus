# 34 运维日报与周报设计

## 目标

把 Hermes 的执行结果自动整理成适合管理和复盘的日报、周报。

## 日报内容

- 在线节点数
- 本日巡检完成数
- 本日变更完成数
- 告警数量
- 失败任务数量
- 回滚数量
- 今日关键事件

## 周报内容

- 设备健康趋势
- 告警趋势
- 任务成功率趋势
- 高风险动作统计
- 常见故障类型
- 经验沉淀摘要

## 自动生成原则

- 数据来自任务和事件流
- 文字摘要由 Hermes 生成
- 关键数字必须可追溯
- 允许附带建议和下一步行动

## 适用对象

- 运维团队
- 站点负责人
- 平台管理员

## 报告格式建议

- 标题
- 本期概览
- 关键指标
- 异常分析
- 建议动作
- 附录：任务和告警列表
