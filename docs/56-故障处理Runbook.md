# 56 故障处理 Runbook

## 目标

把常见故障处理方式写成可直接执行的运行手册。

## 常见故障

### 1. 节点离线
处理：
- 检查网络
- 检查证书
- 检查节点进程
- 检查云端心跳

### 2. 任务卡住
处理：
- 查看步骤状态
- 检查适配器日志
- 检查重试次数
- 必要时取消并回滚

### 3. 设备认证失败
处理：
- 检查密钥是否过期
- 检查权限是否变更
- 检查设备侧配置

### 4. 告警无法闭环
处理：
- 检查事件流
- 检查关联任务
- 检查工单状态
- 检查是否需要人工介入

## Runbook 格式

每个故障条目应包含：
- 现象
- 可能原因
- 排查步骤
- 修复动作
- 验证方法
- 回滚方法

## 设计原则

- 先止血，再定位
- 低风险操作优先自动化
- 高风险操作必须人工确认
- 处理记录要回写知识库
