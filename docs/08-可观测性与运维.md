# 08 可观测性与运维

## 需要观测什么

- 节点在线状态
- 设备在线率
- 任务成功率
- 任务延迟
- 回滚次数
- 重试次数
- 告警频率

## 推荐组件

- Metrics：Prometheus / VictoriaMetrics
- Logs：Loki
- Traces：OpenTelemetry / Jaeger
- Dashboards：Grafana

## 观测对象

### 云端
- API 延迟
- 队列堆积
- 审批耗时
- 任务调度成功率

### 边缘
- Hermes 节点心跳
- 本地队列长度
- 设备适配器可用性
- 离线缓存积压

### 设备侧
- CPU / 内存 / 磁盘
- 端口 / 接口 / 服务状态
- 固件版本
- 配置漂移

## 运维建议

- 建立区域级健康面板
- 任务执行必须有 trace_id
- 重要动作做告警分级
- 云端和边缘分别保留本地日志
