# 72 开工前 10 条清单

## 目标

把 HermesNexus 正式开工前最关键的 10 个条件压缩成一页，便于你在开始 MVP 开发前快速确认。

## 开工前必须全部满足

- [ ] 1. 仓库目录结构已确定
- [ ] 2. cloud / edge / shared / console / deploy / tests / docs 主目录已就位
- [ ] 3. HermesNexus 使用独立虚拟环境
- [ ] 4. HermesNexus 与当前主 Hermes 环境隔离
- [ ] 5. 已明确项目级 `HERMES_HOME`
- [ ] 6. 开发部署测试服务器已可用：`scsun@172.16.100.101:22`
- [ ] 7. Docker / Docker Compose 可用
- [ ] 8. 云端 / 边缘 / 前端技术栈已定
- [ ] 9. `.env.example` 和 `config.example.yaml` 已有
- [ ] 10. 一键启动 / 测试 / 清理命令已定义

## 开工判定

- 全部勾选：可以进入 MVP 开发
- 任意一项未勾选：先补齐前置条件，不进入实现
