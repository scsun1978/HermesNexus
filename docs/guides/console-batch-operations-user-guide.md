# HermesNexus 控制台批量操作用户指南

## 📋 概述

HermesNexus v1.2 控制台现在支持批量操作功能，允许用户通过Web界面一次性管理多个资产和任务。

## 🎯 主要功能

### 1. 批量选择
- **单个选择**: 点击每行的复选框选择单个资产
- **全选功能**: 点击表头复选框一键选择当前页所有资产
- **选择计数**: 批量操作按钮实时显示已选择的资产数量

### 2. 批量操作类型

#### 批量更新
- 更新资产状态（活跃/非活跃/已退役）
- 修改资产描述
- 添加资产标签
- 支持遇错停止选项

#### 批量删除
- 一次性删除多个资产
- 提供确认提示防止误操作
- 详细的删除结果反馈

#### 批量停用
- 批量设置资产为非活跃状态
- 适用于临时停用多个设备

## 🚀 使用步骤

### 步骤1: 选择资产
1. 进入资产管理页面
2. 使用复选框选择要操作的资产
3. 可以点击表头复选框全选当前页

### 步骤2: 启动批量操作
1. 选择资产后，点击"批量操作 (N)"按钮
2. 在弹出的对话框中选择操作类型

### 步骤3: 配置操作参数
根据选择的操作类型，配置相应参数：
- **批量更新**: 选择状态、描述、标签等
- **批量删除**: 确认删除操作
- **批量停用**: 确认停用操作

### 步骤4: 执行操作
1. 点击"执行操作"按钮
2. 等待操作完成
3. 查看操作结果

### 步骤5: 查看结果
操作结果包含：
- **操作摘要**: 总数、成功、失败、成功率
- **详细结果**: 每个资产的操作状态
- **错误信息**: 失败资产的详细错误
- **导出功能**: 可导出JSON格式结果

## 📊 结果解读

### 摘要信息
```
操作ID: asset-update-batch-12345
状态: ✅ 完成
总数: 10
成功: 8
失败: 2
成功率: 80.0%
```

### 错误摘要
```
错误摘要:
- 验证错误: 1 个
- 未找到错误: 1 个
```

### 详细结果
- ✅ **成功项**: 绿色背景，显示成功标记
- ❌ **失败项**: 红色背景，显示错误信息和错误代码

## ⚠️ 注意事项

### 操作限制
- 单次批量操作最大100个资产
- 批量操作执行时间可能较长，请耐心等待
- 操作执行期间请勿关闭浏览器

### 数据安全
- 批量删除操作不可撤销，请谨慎操作
- 建议先执行批量更新测试操作
- 重要操作前建议备份数据

### 错误处理
- 部分失败不影响其他资产的操作
- 可查看详细错误信息了解失败原因
- 支持"遇错停止"选项控制错误处理行为

## 🔧 高级功能

### 结果导出
1. 在操作结果页面点击"导出结果"按钮
2. 下载JSON格式的操作结果文件
3. 可用于后续分析或审计

### 过滤选择
- 结合页面过滤功能选择特定类型的资产
- 支持按状态、类型、搜索条件过滤后批量操作

### 操作历史
- 所有批量操作都会记录操作ID
- 可通过操作ID查询操作详情
- 支持操作追踪和审计

## 📱 界面说明

### 批量操作按钮
- **位置**: 页面右上角
- **显示**: 仅在有选中资产时显示
- **格式**: "批量操作 (数量)"

### 复选框
- **表头复选框**: 全选/取消全选当前页
- **行复选框**: 选择/取消选择单个资产

### 操作对话框
- **简洁设计**: 清晰的操作选项
- **即时验证**: 参数输入实时验证
- **确认机制**: 重要操作二次确认

## 🎨 用户界面

### 配色方案
- **批量操作**: 紫色主题 (与单项操作区分)
- **成功状态**: 绿色背景
- **失败状态**: 红色背景
- **警告信息**: 黄色边框

### 响应式设计
- 支持桌面浏览器
- 支持平板设备
- 移动端基础功能可用

## 🚀 最佳实践

1. **分批操作**: 大量资产建议分批操作
2. **测试先行**: 重要操作先在小范围测试
3. **结果检查**: 操作完成后仔细检查结果
4. **错误处理**: 根据错误信息修正问题后重试
5. **数据备份**: 定期备份重要资产数据

## 📞 技术支持

如遇问题，请：
1. 查看操作结果的错误信息
2. 检查浏览器控制台是否有错误
3. 联系系统管理员协助处理
4. 查看系统日志了解详细错误
