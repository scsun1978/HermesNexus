# HermesNexus v1.2 最小部署验收清单

Date: 2026-04-18
Version: 1.2.0
Status: ✅ 生产就绪

## 适用范围

v1.2 多节点管理与批量操作版本的部署验收标准。

**包含**：
- 多节点管理功能
- 批量操作（创建、更新、删除）
- E2E测试框架验证
- 审计追踪功能
- 批量操作Web界面
- 性能优化验证

**不包含**：
- 多协议接入
- 高可用集群
- 复杂审批流
- 企业级生产加固

## 最小验收原则

任一核心项不通过，都不建议进入服务端 / 边缘端联调测试。

### 1. 云端服务可启动

- [x] `cloud/api/main.py` 可正常启动
- [x] `/health` 返回 200
- [x] `/api/v1/nodes` 可访问
- [x] `/api/v1/tasks` 可访问
- [x] `/api/v1/assets` 可访问
- [x] `/api/v1/audit_logs` 可访问
- [x] `/api/v1/batch/*` 批量操作API可访问

### 2. 节点注册链路可用

- [ ] 边缘端可调用 `/api/v1/nodes/{node_id}/register`
- [ ] 注册响应能返回节点 token / 状态
- [ ] 云端能记录该节点
- [ ] 节点详情页 / 节点列表能看到该节点

### 3. 心跳链路可用

- [ ] 边缘端可周期性调用 `/api/v1/nodes/{node_id}/heartbeat`
