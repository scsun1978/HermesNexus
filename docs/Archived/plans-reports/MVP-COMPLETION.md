## 🚀 快速开始

### 启动系统
```bash
# 1. 启动云端API
make run-cloud

# 2. 启动边缘节点 (可选)
make run-edge

# 3. 访问控制台
# http://localhost:8080/console
```

### 验证系统
```bash
# 健康检查
curl http://localhost:8080/health

# 系统状态
python tests/scripts/system_health_check.py

# 运行测试
python -m pytest tests/unit/test_shared_modules.py -v
```
