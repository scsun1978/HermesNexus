# HermesNexus MVP 发布验收清单

## 📋 验收概览

**项目**: HermesNexus 分布式边缘设备管理系统
**版本**: MVP v1.0
**目标**: 实现基础的云端创建任务 → 边缘节点接收 → SSH 执行 → 结果回传 → 云端可见

---

## ✅ 功能验收

### 核心流程 (必须全部通过)

- [ ] **任务创建与分配**
  - [ ] 可以通过API创建任务
  - [ ] 任务能够分配到在线的边缘节点
  - [ ] 任务包含目标设备、命令、超时等基本信息

- [ ] **边缘节点执行**
  - [ ] 边缘节点能够接收任务分配
  - [ ] 节点能够通过SSH连接到目标设备
  - [ ] 能够执行基本命令（如 `uptime`、`hostname`）
  - [ ] 执行结果能够被正确捕获

- [ ] **结果回传与存储**
  - [ ] 执行结果能够回传到云端
  - [ ] 结果包含stdout、stderr、退出码
  - [ ] 云端能够存储和查询任务结果

- [ ] **状态可见性**
  - [ ] 可以通过控制台查看节点状态
  - [ ] 可以查看任务列表和状态
  - [ ] 可以查看事件日志
  - [ ] 可以查看系统统计信息

### 扩展功能 (至少80%通过)

- [ ] **节点管理**
  - [ ] 节点注册和心跳
  - [ ] 节点状态监控（CPU、内存使用率）
  - [ ] 节点上线/离线状态更新

- [ ] **错误处理**
  - [ ] SSH连接失败处理
  - [ ] 命令执行失败处理
  - [ ] 任务超时处理
  - [ ] 错误日志记录

- [ ] **审计和日志**
  - [ ] 操作审计日志
  - [ ] 事件日志记录
  - [ ] 日志查询和过滤
