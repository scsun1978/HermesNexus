# HermesNexus MVP 开发执行计划

## 📋 MVP 范围确认 (Phase 1)

基于文档66和67的要求，锁定以下MVP边界：

### ✅ 包含范围
- **1个云端控制平面** - 基础任务管理API
- **1个边缘节点** - 简单运行时
- **Linux主机** - 通过SSH管理
- **基础任务类型** - 命令执行
- **最小闭环** - 云端创建任务 → 边缘接收 → SSH执行 → 结果回传 → 云端展示

### ❌ 明确排除
- 多协议设备（仅SSH）
- 多设备类型（仅Linux主机）
- 高可用集群
- 复杂权限体系
- 完整控制台
- 生产级监控
