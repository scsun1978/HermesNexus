- **config.example.yaml** - YAML 配置文件模板，包含：
  - 全局配置
  - 云端服务详细配置
  - 边缘节点详细配置
  - 任务和设备管理配置

  - MVP 限制配置

- **Makefile** - 统一的开发命令接口，支持：

  - 清理工具 (clean, clean-all)

- **deploy/scripts/deploy.sh** - 自动化部署脚本，功能包括：
