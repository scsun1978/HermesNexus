## ✅ 完成剩余项目的快速步骤

### 如果选择 Python 技术栈:

1. **创建 Python 虚拟环境**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Linux/Mac
   # 或 venv\Scripts\activate  # Windows
   ```

2. **设置项目级 HERMES_HOME**
   ```bash
   # 在 .env 中添加
   HERMES_HOME=.hermes
   HERMES_PROJECT=hermesnexus
   ```

3. **创建 requirements.txt**
   ```txt
   # 云端依赖
   fastapi==0.104.1
   uvicorn==0.24.0
   sqlalchemy==2.0.23
   psycopg2-binary==2.9.9
   redis==5.0.1
   pydantic==2.5.0
   python-dotenv==1.0.0

   # 边缘依赖
   aiohttp==3.9.1
   paramiko==3.4.0
   ```

4. **更新 Makefile**
   添加虚拟环境激活和依赖安装命令
